      16                 :             :  * 
      17                 :             :  * @section theory Physical Theory
      18                 :             :  * 
      19                 :             :  * The thermal model is based on the thermal-electrical analogy:
      20                 :             :  * - Temperature (T) ↔ Voltage (V)
      21                 :             :  * - Heat flow (Q) ↔ Current (I)
      22                 :             :  * - Thermal resistance (Rth) ↔ Electrical resistance (R)
      23                 :             :  * 
      24                 :             :  * Key equations:
      25                 :             :  * - Conduction: R_th = L / (k × A)  [K/W]
      26                 :             :  *   where L = length, k = thermal conductivity, A = cross-sectional area
      27                 :             :  * 
      28                 :             :  * - Convection: R_th = 1 / (h × A)  [K/W]
      29                 :             :  *   where h = convection coefficient [W/(m²·K)]
      30                 :             :  * 
      31                 :             :  * - Natural convection coefficient (Churchill & Chu correlation):
      32                 :             :  *   Nu_L = {0.825 + 0.387 Ra_L^(1/6) / [1 + (0.492/Pr)^(9/16)]^(8/9)}²
      33                 :             :  *   h = Nu_L × k_air / L
      34                 :             :  * 
      35                 :             :  * - Radiation: R_th = 1 / (h_rad × A)
      36                 :             :  *   where h_rad = ε × σ × (T_s² + T_a²)(T_s + T_a)  [Stefan-Boltzmann law]
      37                 :             :  * 
      38                 :             :  * @section architecture Architecture
      39                 :             :  * 
      40                 :             :  * The system uses a node-based thermal network:
      41                 :             :  * 
      42                 :             :  * 1. **ThermalNetworkNode**: Represents discrete temperature points
      43                 :             :  *    - Core nodes: Central column, yokes, toroidal segments
      44                 :             :  *    - Turn nodes: Individual windings with quadrant subdivision
      45                 :             :  *    - Ambient node: Reference temperature boundary
      46                 :             :  * 
      47                 :             :  * 2. **ThermalNodeQuadrant**: Subdivision of turn surfaces for directional heat transfer
      48                 :             :  *    - TOROIDAL: RI (Radial Inner), RO (Radial Outer), TL (Tangential Left), TR (Tangential Right)
      49                 :             :  *    - CONCENTRIC: L (Left), R (Right), T (Top), B (Bottom)
      50                 :             :  * 
      51                 :             :  * 3. **ThermalResistanceElement**: Connections between node quadrants
      52                 :             :  *    - Type: CONDUCTION, NATURAL_CONVECTION, FORCED_CONVECTION, RADIATION
      53                 :             :  *    - Calculated based on geometry and material properties
      54                 :             :  * 
      55                 :             :  * @section algorithm Connection Algorithm
      56                 :             :  * 
      57                 :             :  * Turn-to-Turn Connections (Purely Geometric):
      58                 :             :  * - Condition: surface_distance < min(wire_width, wire_height) / 4
      59                 :             :  * - Surface distance = center_distance - extent1 - extent2
      60                 :             :  * - Extent = min(wire_dimensions) / 2 for each wire
      61                 :             :  * - Facing quadrants determined by dot product of direction vectors
      62                 :             :  * 
      63                 :             :  * Convection to Air (Purely Geometric):
